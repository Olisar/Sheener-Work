# SHEEner - EHS Performance Management

SHEEner is a comprehensive management system for Environment, Health, and Safety (EHS) KPIs and reporting.

## 🚀 Quick Start
- **Environment**: Requires XAMPP (Apache/PHP 8.0+ and MySQL).
- **Deployment**: Clone this repository into your `htdocs` folder.
- **Access**: Navigate to `http://localhost/sheener` in your browser.

## 📖 Documentation
- **[Architecture Guide](ARCHITECTURE.md)**: Detailed explanation of how the frontend and backend are structured.
- **[Sync Instructions](SyncSheener.bat)**: Use the desktop shortcut to sync changes to GitHub.

## 📂 Key Directories
- `/css`: Stylesheets.
- `/js`: JavaScript logic.
- `/php`: Backend API.
- `/includes`: Reusable components.
